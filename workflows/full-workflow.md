# Full Workflow — LOCUS

**Tier:** Full · 7 prompts · ~45 minutes
**Best for:** Standard client dossiers, listing analysis, buyer advisory, most residential intelligence engagements

---

## When to Use Full

| Situation | Use Full? |
|-----------|----------|
| Client-facing dossier deliverable | ✅ Yes |
| Residential buyer advisory | ✅ Yes |
| Listing competitive analysis | ✅ Yes |
| Landlord/tenant site selection | ✅ Yes |
| Detailed green space / education analysis needed | ❌ No — use Complete |
| Investment decision with full data | ❌ No — use Complete |
| Multi-address portfolio | ❌ No — use Enterprise |

## Execution Sequence

### Phase 1 — Foundation (run first, required)

**→ P01 — Walkability & Geospatial** (`prompts/P01-walkability-geospatial.md`)

This is the most important prompt in the sequence:
- Establishes geocoded coordinates (required by all subsequent bash API calls)
- Collects Walk Score®, Transit Score®, Bike Score® (required by scoring-agent)
- Sets FIPS codes (required by P07 and P08)

*Do not proceed to Phase 2 until P01 output file exists.*

---

### Phase 2 — Lifestyle Data (can run in any order)

**→ P02 — Food, Drink & Social** (`prompts/P02-food-drink-social.md`)
Focus: Restaurant density, cuisine diversity, coffee, cultural venues, nightlife.

**→ P04 — Daily Errands & Retail** (`prompts/P04-daily-errands-retail.md`)
Focus: Grocery access, pharmacy, healthcare, general retail.

**→ P06 — Transportation & Commute** (`prompts/P06-transportation-commute.md`)
Focus: Transit routes/frequency, CBD commute time, highway access, airport.

---

### Phase 3 — Civic & Market Data

**→ P07 — Civic Infrastructure & Safety** (`prompts/P07-civic-infrastructure.md`)
*Fair Housing sensitivity — follow protocol in prompt.*
Focus: Crime rates (objective), flood zone, AQI, noise.

**→ P08 — Demographics & Trajectory** (`prompts/P08-demographics-trajectory.md`)
*Fair Housing sensitivity — economic indicators only.*
Focus: Population change, income change, home value appreciation, permits.

---

### Phase 4 — Synthesis

**→ P10 — Synthesis & Final Dossier** (`prompts/P10-synthesis-dossier.md`)

Requires all Phase 1–3 prompts complete. Runs scoring-agent and narrative-agent.
Output: `final-dossier.md`, `scorecard.md`, `day-in-the-life.md`

## Upgrade to Complete

To upgrade a Full engagement to Complete (add P03, P05, P09):

1. Run P03 — Recreation & Green Space
2. Run P05 — Education (skip for Investor persona)
3. Run P09 — Business Environment
4. Re-run P10 — scoring-agent will incorporate new data; narrative-agent will expand dossier
